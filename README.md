# Mood Tracker

A Personalized Mood Tracker and Activity Suggestion Platform.

## Features
- Track your daily moods.
- Get activity suggestions based on your mood.
- Simple and easy to use.

## Setup
1. Clone the repository.
2. Run `npm install`.
3. Start the server with `npm start`.

## Technology Stack
- Node.js
- Express
- HTML/CSS/JS
